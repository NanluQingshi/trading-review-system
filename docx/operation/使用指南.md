# 交易复盘统计系统 - 使用指南

## 📋 项目简介

这是一个本地化的交易复盘和数据统计平台，帮助您记录、分析和优化交易策略。

### 核心功能

1. **Method 库管理** 📚

   - 创建和管理交易方法
   - 记录每个方法的规则和描述
   - 追踪每个方法的胜率

2. **交易复盘** 📊

   - 记录每笔交易的详细信息
   - 关联使用的交易方法
   - 添加交易笔记和标签

3. **我的统计** 📈
   - 可视化展示交易数据
   - 多维度统计分析
   - 盈亏曲线追踪

## 🚀 快速开始

### 首次安装

```bash
# 进入项目目录
cd trading-review-system

# 安装所有依赖（根目录、前端、后端）
npm run install-all
```

### 启动应用

#### 方式一：同时启动前后端（推荐）

```bash
npm run dev
```

#### 方式二：分别启动

```bash
# 终端1 - 启动后端
npm run server

# 终端2 - 启动前端
npm run client
```

### 访问地址

- **前端界面**: http://localhost:3000
- **后端 API**: http://localhost:5050/api

## 📖 功能使用说明

### 1. Method 库管理

**添加新方法：**

1. 点击右上角"添加新方法"按钮
2. 填写方法名称、分类、描述和交易规则
3. 点击确定保存

**编辑方法：**

- 点击方法卡片上的编辑图标
- 修改信息后保存

**删除方法：**

- 点击方法卡片上的删除图标
- 确认删除

### 2. 交易复盘

**添加交易记录：**

1. 点击"添加交易记录"按钮
2. 填写交易详情：
   - 货币对（如：EUR/USD）
   - 交易方向（做多/做空）
   - 入场价格和出场价格
   - 手数
   - 入场时间和出场时间
   - 使用的方法
   - 交易结果
   - 风险回报比
   - 标签和笔记

**查看和管理：**

- 表格展示所有交易记录
- 可以编辑或删除任何记录
- 支持按多个维度筛选

### 3. 我的统计

**核心指标：**

- 总交易次数
- 胜率
- 总盈亏
- 盈亏因子

**可视化图表：**

- 累计盈亏曲线
- 交易结果分布（饼图）
- 各方法盈亏对比（柱状图）

**详细统计：**

- 按货币对统计
- 按方法统计
- 每日盈亏追踪

## 💾 数据说明

### 当前数据模式

系统目前使用**模拟数据**进行开发和测试，包含：

- 4 个预设交易方法
- 8 条示例交易记录

### 数据持久化

**临时存储：**

- 当前数据存储在内存中
- 重启服务器后会重置为初始模拟数据

**未来升级：**

- 可以轻松切换到 SQLite 数据库
- 实现真正的数据持久化

## 🔧 技术架构

### 前端

- **框架**: React 18 + TypeScript
- **UI 库**: Ant Design
- **图表**: Recharts
- **路由**: React Router
- **HTTP**: Axios

### 后端

- **运行时**: Node.js
- **框架**: Express
- **数据库**: SQLite3（准备就绪）
- **当前**: 使用模拟数据

## ❓ 常见问题

**Q: 数据会丢失吗？**
A: 当前版本使用内存存储，重启后会重置。后续会升级到数据库持久化。

**Q: 如何备份数据？**
A: 当前版本暂不支持。数据库版本会支持导出功能。

**Q: 可以多人使用吗？**
A: 当前是单机版本，适合个人使用。

**Q: 如何修改端口？**
A: 修改 `server/index.js` 中的 PORT 变量和 `client/src/services/api.ts` 中的 API_BASE_URL。
